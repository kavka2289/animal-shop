## Куда класть фото товаров

Складывай изображения карточек сюда:

`shop/static/shop/img/products/`

## Как назвать файлы

Название файла должно совпадать со значением `image` в `shop/data.py`.

Пример:

- если в товаре `image: "dog-food"`, то файл должен быть `dog-food.png`

Сейчас в проекте используются такие ключи:

- `dog-food.png`
- `leash.png`
- `litter.png`
- `toy-mouse.png`
- `bird-seed.png`
- `hamster.png`

Можно использовать свои картинки — просто замени/добавь нужные файлы с такими именами.
